# text2blog

## 프로젝트 주제

정리되지 않은 기술 텍스트와 논문 자료를 AI가 블로그 글과 시각 자료로 자동 변환해주는 기술 콘텐츠 제작 플랫폼입니다.

## 프로젝트 개발 배경

기술 블로그나 연구 콘텐츠를 작성할 때는 원본 자료를 읽고, 핵심 내용을 구조화하고, 글의 흐름을 만들고, 필요한 이미지를 따로 제작해야 하는 불편함이 있습니다.

특히 논문 기반 글이나 개발 트러블슈트 글은 정보량이 많고 전문 용어가 많아서 초안 작성에 많은 시간이 걸립니다. 본 프로젝트는 이러한 과정을 줄이고, 기술 자료를 더 빠르게 블로그 콘텐츠로 전환하기 위해 만들어졌습니다.

## 프로젝트의 주요 기능

### 1. 텍스트 기반 Figure 생성

논문 PDF, 초록 등 raw 텍스트를 입력하면 주요 개념과 관계를 분석해 다이어그램, 파이프라인 프레임워크, comparison table을 생성합니다.

### 2. 기술 트러블슈트 솔루션 작성

디버깅 과정과 작업 메모를 입력하면 문제 상황, 분석, 해결 과정, 결과 형식의 기술 블로그 초안을 생성합니다.

### 3. 블로그 품질 검수

생성된 글과 이미지가 원본 내용을 잘 반영하는지, 이미지가 글의 흐름에 자연스럽게 들어가는지, 메타데이터가 적절한지 검수하는 구조를 가집니다.

## 에이전트 역할

- `text-to-blog-generator`: 원본 텍스트를 PAAR 구조로 분석해 한국어 기술 블로그 초안과 이미지 슬롯 명세를 생성하는 에이전트.
- `text-to-image-generator`: 블로그의 이미지 슬롯 명세를 바탕으로 썸네일, 다이어그램, 차트 등 시각 자료를 생성하는 에이전트.
- `content-reviewer`: 생성된 블로그, 이미지, 통합본이 원본 내용과 맞고 글 흐름에 자연스럽게 연결되는지 검수하는 에이전트.

## 실행 방법

```bash
npm install
npm run dev
```

개발 서버 실행 후 브라우저에서 `http://localhost:3000`으로 접속합니다.
